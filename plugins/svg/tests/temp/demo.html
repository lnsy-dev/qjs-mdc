CSS-Styled SVG Charts Demo

Default Monochrome Style

Generated with --externally-styled flag, using default CSS variables.

Custom Colored Style

Same SVG, different CSS! Pattern colors and strokes are customized via CSS variables.

How to Use

  1. Generate an externally-styled SVG: 
    ./generate-svg.js data.json --externally-styled > chart.svg
  2. Generate the CSS template: 
    ./tests/generate-css.js > styles.css
  3. Customize the CSS variables in styles.css
  4. Link the CSS to your SVG or embed it in HTML

Available CSS Classes

  • .data-element - All data visualization elements (bars, circles, paths)
  • .data-index-N - Specific data element by index (e.g., .data-index-0)
  • .data-category-{name} - Elements grouped by category (e.g., .data-category-A)
  • .axis - Chart axes
  • .label - Text labels

Available CSS Variables

  • --pattern-fill - Background color for patterns
  • --pattern-stroke - Stroke color for patterns
  • --data-element-fill - Fill for data elements
  • --data-element-stroke - Stroke for data elements
  • --data-element-stroke-width - Stroke width for data elements
  • --axis-stroke - Stroke color for axes
  • --axis-stroke-width - Stroke width for axes
  • --label-fill - Text color for labels
  • --label-font-size - Font size for labels
