🗺️ SVG Map Generator Demo

About: This demo showcases the SVG map generator's capabilities including GeoJSON rendering, custom icons, and various geometry types.

1. Basic Map (Raw GeoJSON with CLI Bounds)

Simple map with Points, LineStrings, and Polygons using CLI bounds flags.

./generate-svg.js map.geojson \
  --nw-lat=38.0 --nw-lon=-123.0 \
  --se-lat=34.0 --se-lon=-118.0 \
  > map.svg

2. Map with Custom Icons

Using config wrapper format with custom SVG icons for different feature types.

{
  "northWestBounds": [38.0, -123.0],
  "southEastBounds": [34.0, -118.0],
  "iconList": {
    "city": "<svg>...</svg>"
  },
  "geojson": { ... }
}

3. Externally Styled Map

Map with CSS classes for external styling (no inline colors).

./generate-svg.js map.geojson --externally-styled > map.svg

Features

  • ✅ All GeoJSON geometry types (Point, LineString, Polygon, Multi*, GeometryCollection)
  • ✅ Custom SVG icons mapped by feature type
  • ✅ Automatic bounds detection or CLI specification
  • ✅ Extensive data attributes for interactivity
  • ✅ Pattern fills for print-friendly output
  • ✅ External CSS styling support
  • ✅ YAML front matter configuration
  • ✅ Background SVG embedding
